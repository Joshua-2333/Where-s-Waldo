# Where’s Waldo

A fun “Where’s Waldo” style hidden object game built with HTML, CSS, and JavaScript.  
Find all characters in each scene as fast as possible, then compete on a leaderboard.

---

## What This Project Is

This is a **full-stack web application** with both a **frontend** and **backend**:

### Frontend
- Built with **HTML, CSS, and vanilla JavaScript**
- Contains the game UI, timer, character selection, and leaderboard modal
- Handles user interactions and sends requests to the backend

### Backend
- Built with **Node.js and Express**
- Provides REST API endpoints for:
  - validating guesses
  - saving scores
  - fetching scene leaderboards
- Uses **PostgreSQL** as the database

---

## Gameplay

1. Choose a scene from the main screen.
2. Click on the image where you think a character is hiding.
3. Select the character from the dropdown.
4. If correct, the character will be marked as found.
5. Find all 4 characters as fast as possible.
6. When you win, your time is saved and the leaderboard appears.
